Latest v0.0.1 Early Alpha
ThePlugNoFap — First Release
A modern, offline desktop streak tracker built with Python 3.14
2025 · Initial
This tool is designed to help you build discipline, improve focus, and stay consistent with your personal goals. It encourages self-control and a better daily routine through a clean, distraction-free experience.
✦ Features
  • ⏱ Smooth Circular Timer — High-quality visual animations for progress tracking, designed to keep you focused and aware of time.
  • 🔒 Offline-first Design — No login or internet connection required. Completely offline — 100% of the time.
  • 🌑 Minimal UI — Clean and lightweight design with zero visual clutter. Just the essentials.
  • 💾 Local Storage — All your data is saved securely on your own device. No cloud, no sync, no exposure.
⚠ Current State — Early Alpha
⚠️
This is an early alpha build. Bugs may exist and some features are still under active development. User feedback is welcome — report issues directly via the GitHub repository. Some UI elements are still placeholders and will be refined in future releases.
🗺 Roadmap
  • 🏆 Ranks & Achievements system — Milestone rewards and badges to celebrate your progress.
  • 🎨 Custom UI improvements — Personalization options for themes, accent colors and iconography.
  • ☁️ Possible cloud features — Optional sync in a future version — always opt-in, never forced.
⬇ Installation
1
Download ThePlugNoFapSetup.exe from the GitHub Releases page.
2
Run the installer — no admin rights required for standard install.
3
Launch the app and start your streak. That's it — no sign-up, no setup wizard.
